Formas de Mostrar uma Data

getFullYear( )

Mostra o ano com os 4 dígitos (yyyy)

getMonth( )

Mostra o mês como um número (0-11)

getDate( )

Mostra o dia como um número (1-31)

getDay( )

Mostra o dia da semana como um número (0-6)

getHours( )

Mostra a hora (0-23)

getMinutes( )

Mostra os minutos (0-59)

getSeconds( )

Mostra os segundos (0-59)

getMilliseconds( )

Mostra os milisegundos (0-999)

getTime( )

Mostra o tempo (em milisegundos desde o dia 1 de janeiro de 1970)
