Métodos Array

pop( )

O método pop( ) remove o último elemento de uma array

push( )

O método push( ) adiciona um novo elemento a uma array (no final dela)

Exemplo:

splice( )

O método splice( ) pode ser usado para adicionar novos itens em uma array

O primeiro parâmetro define a posição onde os novos elementos serão adicionados.

O segundo parâmetro define quantos elementos deverão ser removidos.

O resto do parâmetro define quais elementos serão adicionados.
