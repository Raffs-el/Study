Funções

Uma função em JavaSript serve para realizar uma determinada tarefa e, para isso, nós precisamos "invocá-la" (chamá-la).

Sintaxe da Função

Uma função em JavaScript é definida com a palavra chave function, seguinda por um nome ao lado de um parênteses ( ).

Os nomes da função podem conter letras, dígitos, palavras sublinhadas e símbolos monetários.

Os parênetes devem incluir nomes de parâmetros separados por vírgulas: (parâmetro1, parâmetro2, ...)

O código a ser executado, pela função, é colocado dentro de chaves: { }

Os parâmetros de uma função são listado dentro dos parênteses ( ) na definição da função.

Os argumentos das funções são os valores recebidos pela função quando ela é invocada.

Dentro da função, os argumentos (parâmetros) comportam-se como variáveis locais.

Invocação da Função

O código dentro da função será executado quando "algo" invocar(chamar) a funçao:

  • Quando um evento ocorre (quando um usuário clica em um botão);
  • Quando ela é invocada (chamada) de um código JavaScript;
  • Automaticamente (invocada por ela mesma).

Função Return (Retornar)

Quando o JavaScript alcança uma declaração return, a função irá parar de ser executada.

Se a função foi invocada de uma declaração, o JavaScript vai "return" (retornar) a execução do código depois da declaração de chamada.

Funções geralmente computam um return value (valor de retorno). O valor de retorno é "retornado" de volta ao "chamador".

Variáveis declaradas dentro uma função JavaSript se tornam restritas àquela função.

Como as variáveis locais são reconhecidas somente dentro de suas funções, variáveis com o mesmo nome podem ser utilizadas em diferentes funções.

Variáveis locais são criadas quando uma função começa, e são deletadas quando a função é finalizada.
