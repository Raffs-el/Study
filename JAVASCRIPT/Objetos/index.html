Métodos de Objetos

Métodos são ações que podem ser realizadas em objetos.

Um método é uma função armazenada como uma propriedade.

A palavra chave this utilizada dentro de uma função refere-se ao "dono" da função, ou seja, se for utilizada uma função dentro de um objeto, o "dono" dela será o próprio objeto.
