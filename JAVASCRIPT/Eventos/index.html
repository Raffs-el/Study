Eventos

Os eventos HTML são "coisas" que ocorrem em elementos HTML.

Quando utilizado em páginas HTML, o JavaScript pode "reagir" a esses eventos.

Eventos HTML

Um evento HTML pode ser algo que o navegador faz, ou alguma coisa que um usuário faz.

Exemplos de eventos HTML:

  • Uma página que acabou de ser carregada
  • Um campo de input que mudou
  • Um botão que foi clicado

Geralmente, um evento acontece quando você quer fazer alguma coisa.

O JavaScript te permite executar o código quando o evento é detectado.

O HTML permite colocar atributos de eventos, com um código javascript, a ser adicionado aos elementos HTML.

Exemplo: botão clicável que mostra o horário

Lista de alguns eventos HTML

onchange = um evento HTML será modificado

onclick = o usuário clica em um elemento HTML

onmouseover = o usuário move o mouse em cima de um elemento HTML

onmouseout = o usuário move o mouse para fora de um elemento HTML

onkeydown = o usuário aperta uma tecla do teclado

onload = quando o browser termina de carregar a página
