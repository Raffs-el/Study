Métodos JavaScript

Convertendo para Upper e Lower Case

Uma string é convertida para upper case com toUpperCase( );

Uma string é convertida para lower case com toLowerCase( ).

Exemplo Upper Case:

olá

Exemplo Lower Case:

OLÁ
