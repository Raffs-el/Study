Strings

Strings são utilizadas para armazenar e manipular textos.

As strings são escritas dentro de aspas (simples ou duplas), podendo conter zero ou mais caracteres.

String length

Para descobrir a extensão de uma string, use a propriedade length.

String slice()

slice() extrai uma parte de uma string e retorna a parte extraída em uma nova string.

O método possui 2 parâmetros: start position e end position.
